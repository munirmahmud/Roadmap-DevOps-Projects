## 🔍 How to Use
1. Clone the repo:
```bash
git clone https://github.com/munirmahmud/roadmapsh-devops-projects.git
```

2. Browse to a project folder:

```bash
cd beginner/01-cli-server-performance
```

3. Read the README and run the script:

```bash
bash script.sh
```
